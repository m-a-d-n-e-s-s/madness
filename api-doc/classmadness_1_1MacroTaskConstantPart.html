Detailed Description

compute the "constant" part of MP2, CC2, or LR-CC2

the constant part is result = G [F,f] |ij> for MP2 result = G [F,f] |t_i t_j> for CC2 result = G [F,f] |t_i x_j> + |x_i t_j> for LR-CC2
