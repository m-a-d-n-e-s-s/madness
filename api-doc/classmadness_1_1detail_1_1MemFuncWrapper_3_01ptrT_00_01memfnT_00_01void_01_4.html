Detailed Description

template<typename ptrT, typename memfnT>
class madness::detail::MemFuncWrapper< ptrT, memfnT, void >

Functor wrapper for object and member function pointers.

This is a specialization of MemFuncWrapper for the case where the member function has void return type.
