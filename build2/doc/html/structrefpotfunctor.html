Detailed Description

reference potential for the Poisson solver of a Gaussian

Given a unnormalized Gaussian return the result of the Poisson equation: [ V(r) = dr' 1/|r-r'| exp(-alpha r'^2) ] Formulas taken from wikipedia: http://en.wikipedia.org/wiki/Poisson%27s_equation#Potential_of_a_Gaussian_charge_density [ ^2 = (r) = Q * exp(-alpha r^2) (r) = Q * 1/r erf(r/{2} ) ] where the Gaussian function is NOT normalized, and alpha = 1/(2 sigma^2)
